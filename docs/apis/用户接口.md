# 用户接口

## 创建用户
- 请求方式: POST
- 请求地址: `/user/`

### 请求参数

```json
{
"username": "",
"password":""  
}
```



## 修改用户
- 请求方式: PUT
- 请求地址: `/user/{{user_id}}`


### 请求参数

```json
{
"username": "",
"password":""  
}
```

## 用户列表
- 请求方式: get
- 请求地址: `/user/query`


### 请求参数
- pageSize: 页面数量
- pageCount: 当前页号


## 绑定部门
- 请求方式: post
- 请求地址: `/user/bind/department/{{user_id}}/{{dept_id}}`

### 请求参数
- user_id: 用户id
- dept_id: 部门id


## 解除部门绑定
- 请求方式: delete
- 请求地址: `/user/bind/un/department/{{user_id}}/{{dept_id}}`
### 请求参数
- user_id: 用户id
- dept_id: 部门id
